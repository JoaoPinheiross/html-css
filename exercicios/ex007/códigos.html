Repósitorio de Códigos HTML

Códigos de Estrura

<!> - É usado para criar o Código inicial.

<link> - Colocar um Favicon.

<h1> - Criar um título.

<hr> - Criar uma linha horizontal

<p> - Criar um parágrafo

<br> - Realizar uma quebra de linha

<img> - Inserir Imagem

Códigos de Formotação

<strong> - Texto em destaque

<em> - Texto em ênfase

<mark> - Marcar texto

<small> - Letra pequena

<del> - Texto deletado

<ins> - Texto inserido

<sup> - texto em cima

<sub> - texto em baixo

<code> - para demonstrar um código

<pre> - Para texto pré fotmatado

<q> - Citações em textos

<blockquote>

Citações à parte do texto

<blockquote cite> - Para link em conjunto com a citação à parte

<abbr> - Abrev.
